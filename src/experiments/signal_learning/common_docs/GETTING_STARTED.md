# 🚀 快速开始指南

## 🎯 你现在就可以开始学习信号机制了！

### 第一步：打开VS Code
1. 在VS Code中打开项目文件夹
2. 导航到 `src/experiments/signal_learning/`

### 第二步：开始调试
1. 按 `Ctrl+Shift+D` 打开调试面板
2. 在顶部选择 "调试实验1: 信号基础观察"
3. 按 `F5` 开始调试

### 第三步：观察信号处理
程序会在信号处理器中暂停，你可以看到：
- 信号编号（11 = SIGSEGV, 8 = SIGFPE, 等）
- 调用堆栈
- 变量值

## 🔍 快速测试不同信号

修改 `launch.json` 中的 `args` 来测试不同信号：

```json
"args": ["1"],  // 空指针解引用
"args": ["3"],  // 除零错误  
"args": ["5"],  // 非法指令
"args": ["7"],  // 程序中止
```

## 📚 在信号处理器中设置断点

在 `01_signal_basics.c` 的这一行设置断点：
```c
printf("🚨 捕获到信号: %d (%s)\n", sig, strsignal(sig));
```

这样你就能观察到信号被触发的那一刻！

## 🎯 学习目标

通过这个实验，你将亲眼看到：

1. **信号就像"软件中断"** - 观察程序如何被打断
2. **SIGSEGV是什么** - 段错误信号的处理过程
3. **信号和异常的关系** - 不同错误产生不同信号
